Vue源码解析18--core.observer.array-对Array的patch

2019-04-26 前端开发

observer.array

Array的以下方法会改变原本的对象，所以需要对其进行patch处理，改造为响应式执行：

  • push
  • pop
  • shift
  • splice
  • sort
  • reverse

本质上是在执行原有API的基础上，添加两个步骤：

  1. 如果有新元素进入，对新元素进行响应式改造
  2. 触发一次更新通知
